# job4j_cars

Учебное web-приложение для размещения объявлений о продаже автомобилей.

## Возможности

- просмотр всех объявлений о продаже автомобилей;
- просмотр подробной страницы объявления;
- регистрация, вход и выход пользователя;
- создание объявления авторизованным пользователем;
- указание характеристик автомобиля: марка, модель, год, тип кузова, двигатель, коробка передач, пробег;
- загрузка фотографии автомобиля;
- статус объявления: продается / продано;
- изменение статуса только автором объявления.

## Стек

- Java 17+
- Spring Boot
- Thymeleaf
- Hibernate
- PostgreSQL или H2 для локальной разработки, если он уже используется в проекте
- Liquibase
- Maven

## Архитектура

Приложение разделено на три слоя:

- `controller` — обработка HTTP-запросов и подготовка данных для Thymeleaf;
- `service` — бизнес-логика: регистрация, вход, создание объявления, проверка автора;
- `repository` — работа с базой данных через Hibernate `SessionFactory`.

Важно: Spring Data в проекте не используется. `SessionFactory` создается как Spring bean и внедряется в репозитории через конструктор.

Основные репозитории:

- `UserRepository`;
- `CarRepository`;
- `PostRepository`.

## Запуск

Проверьте настройки подключения к базе данных в конфигурационных файлах проекта.

Запуск приложения:

```bash
mvn spring-boot:run
```

Запуск тестов:

```bash
mvn test
```

После запуска приложение доступно по адресу:

```text
http://localhost:8080/posts
```
